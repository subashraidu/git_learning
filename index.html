Git Essentials


git init: Initializes a new Git repository in the current directory. git clone : Clones a remote repository to your local machine. git status: Shows the current state of your working directory and staging area. git add : Adds changes from the working directory to the staging area in preparation for a commit. git commit -m "Commit message": Records changes from the staging area to the repository with a descriptive commit message. git pull origin : Fetches changes from a remote repository and merges them into the current branch. git push origin : Pushes committed changes to a remote repository. git branch : Creates a new branch with the specified name. git checkout : Switches to the specified branch. git switch : Switches to the specified branch (Git version 2.23 and later). git checkout -b : Creates and switches to a new branch in a single command. git switch -c : Creates and switches to a new branch in a single command (Git version 2.23 and later). git merge : Combines changes from the specified branch into the current branch. git log: Displays a log of all commits in the repository. git checkout -- : Discards changes in the working directory for the specified file(s). git reset HEAD^: Undoes the last commit while keeping changes in the working directory. git reset --hard HEAD^: Undoes the last commit and discards changes in the working directory. git config --global user.name "Your Name" and git config --global user.email "your.email@example.com": Sets up your name and email globally for Git. git remote -v: Lists all remote repositories associated with the current repository. git diff: Shows the differences between the working directory and the last commit. git tag -a -m "Tag message": Creates an annotated tag for a specific commit. git clean -df: Removes untracked files and directories from the working directory.
